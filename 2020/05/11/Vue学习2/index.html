12.路由嵌套

嵌套路由就是路由中的路由，路径的子路径，如图中/user/foo下有两个profile、posts子路由。
嵌套路由有什么用？可以使用相同模版，网页框架不变，只改变一小部分内容

/user/foo/profile /user/foo/posts
+——————+ +—————–+
| User | | User |
| +————–+ | | +————+ |
| | Profile | | +——-> | | Posts | |
| | | | | | | |
| +————–+ | | +————+ |
+——————+ +—————–+
