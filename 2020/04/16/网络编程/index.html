1.1 网络通信

Tcp/Ip参考模型

1.网络编程中的要素

IP和端口号 IP

网络通信协议 udp ,tcp

1.2 IP

ip地址 ：InetAddress

​ 唯一定位一台网络上的计算机

​ 127.0.0.1 ：本机localhost

​ ip地址分类

​ IPV4/IPV6

​ IPV4 ：127.0.0.1 4个字节 0~255

​ IPV6 : fe80::b058:bbcd:4905:c659%3 128位 8个无符号

​ 公网（互联网）-私网（局域网）

​ ABCD类地址

​ 192.168.XX.Xx专门给组织内部使用的

1.3 端口

端口表示计算机上的一个程序的进程

​ 不同的进程有不同的端口号，用来区分软件

​ 被规定0~65535

​ 公有端口0~1023

​ HTTP:80

​ HTTPS：443

​ FTP:21

​ Telent:23

​ 程序注册端口号

​ Oracle:1521

1.4 通信协议

网络通信协议：速率 传输码率 代码结构 传输控制

大事化小：分层

TCP/IP协议簇 ：实际上是一组协议

​ TCP:用户传输协议

​ UDP:用户数据报协议
